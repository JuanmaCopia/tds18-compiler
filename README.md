HOW TO BUILD THE COMPILER:
    
    Write on console:
    
        git clone https://github.com/JuanmaCopia/tds18-compiler
    
        cd tds18-compiler
      
        ./make.sh
    
    Your compiler should be builded and now you can compile .tds source code
    by running:
        ./c-tds FILENAME.tds

HOW TO RUN TESTS:
    
    ERROR TESTS:
    ./run_compilation_errors_test.sh
    
    FINAL TESTS:
    ./run_final_tests.sh
